## Sponsored Transaction Blocks

The transaction block builder can support sponsored transaction blocks by using the
`onlyTransactionKind` flag when building the transaction block.

```tsx

    const txb = new TransactionBlock();

    // ... add some commands...

    const kindBytes = await txb.build({client: client, onlyTransactionKind: true});

    // construct a sponsored transaction from the kind bytes

    const tempTx = TransactionBlock.fromKind(kindBytes);
    tempTx.setSender(userAddress);
    tempTx.setGasOwner(sponsorAddress);

    //Collect signature from Sponsor
    let sponsoredTx =  await sponsorKeypair.signTransactionBlock(await tempTx.build({client: client}));

        //Collect signature from Sender
    let signedTX = await userKeypair.signTransactionBlock(
        await TransactionBlock.from(sponsoredTx.bytes).build({client: client}));

    client.executeTransactionBlock({
        transactionBlock: signedTX.bytes,
            signature: [signedTX.signature, sponsoredTx.signature],
            requestType: "WaitForLocalExecution",
            options: {...}
        }
    )
```
